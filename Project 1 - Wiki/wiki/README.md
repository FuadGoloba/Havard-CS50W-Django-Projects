# Project1: Wiki

### Description:

Design of Wikipedia-like online encyclopedia. Uses Github Markdown syntax to write the contents of encyclopedia.


#### Features:
* Create New page
* Search for a page
* Edit existing page
* Show random page
* Markdown to HTML Conversion.
